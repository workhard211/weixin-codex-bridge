# Changelog

## 0.1.0

- Added built-in Weixin bot QR login through `npm run login`; new users do not need to install OpenClaw.
- Added `npm run init` to create or update `.env` with beginner-friendly defaults.
- Added Windows setup helper `npm run setup` for dependency install, initialization, preflight, optional login, and startup.
- Added setup preflight checks for Node/npm, Codex Desktop, Weixin credentials, ports, scripts, and local console health.
- Added Desktop UI delivery safeguards for DPI-aware composer detection, calibration, and diagnostic screenshots.
- Added public repository hygiene checks, CI workflow, issue templates, security policy, and open-source checklist.
